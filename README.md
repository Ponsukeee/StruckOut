# StruckOut

HTC VIVE用VRゲームのスクリプト部分。\
Releaseにゲームの実行ファイルを置いています。

画像をクリックするとYouTubeのプレイ動画に飛びます。\

## 使用したアセット
* SteamVR v1.2.3
* NatureStarterKit2
* Ground textures pack
* PostProcessing
* TextMesh Pro
* SixenseInput
* ShaderForge
